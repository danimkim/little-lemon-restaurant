## Demo

https://your-little-lemon-restaurant.vercel.app/

## Installation

1. Clone the repo

```
git clone https://github.com/danimkim/little-lemon-restaurant.git
```

2. Install packages

```
yarn add
```

3. Run the project

```
yarn start
```
